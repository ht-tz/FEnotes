# Lottery  wheel

转盘需求：

1. 服务端给定固定的获奖内容，转盘指针需要转到对应的内容上
2. 转盘上的字，下半部分的文案需要倒转展示，否则用户无法方便的看到奖品内容
3. 根据2的需求，我们这次的转盘只设置，指针转动，但是转盘不转动
4. 转盘的图片基本使用静态图片，但是宫格需要根据css搭建，方便放置奖品内容
5. react 实现
